# Projet
# Analyse des Produits et des Ventes

Ce projet consiste en une **analyse exploratoire des données produits** d'un fichier csv sur les ventes de différents produits Amazon.
Lien du dataset disponible sur Kaggle: https://www.kaggle.com/datasets/spypsc07/amazon-products

L'objectif est de **comprendre les tendances dans les données**, détecter des corrélations et visualiser les distributions par catégorie.
C'est une première étape pour moi dans l'optique de maîtriser des concepts de l'IA, pour l'instant je ne fais que manipuler un fichier csv mais plus tard je serais capable de créer des prédictions par exemple ou autres trucs avancés et cool que la puissance des bibliothèques permettent.

---

## Description du projet

- Étude de la **distribution des prix et des notes** des produits
- Analyse de la **corrélation entre le prix, le volume de ventes et la note moyenne**
- Visualisations : histogramme, scatter plot, box plot, heatmap
- Nettoyage et transformation des données (gestion des valeurs manquantes, conversion des volumes de ventes)

---

## Comment exécuter le projet

1. Cloner le dépôt :

```bash
git clone https://github.com/<votre-username>/Projet.git
````

2. Installer les dépendances (si nécessaire) :

```bash
pip install pandas matplotlib seaborn numpy
```

3. Ouvrir le fichier sur un Notebook, j'utilise personnellement Jupyter Lab mais Google Collab par exemple marche parfaitement :

```bash
jupyter notebook analyse_produits.ipynb
```

4. Exécuter les cellules dans l'ordre pour reproduire l'analyse et les visualisations.
